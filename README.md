# Simple Interest Calculator

A calculator that calculates simple interest given principal, annual rate of interest, and time period in years.

This project is a starting point for the transition of our micro-finance startup's code to a modern version control system (Git/GitHub). It serves as a public example of our commitment to transparency and community involvement as we empower low-income individuals.

## Input & Output

**Input:**
* `p`, principal amount
* `t`, time period in years
* `r`, annual rate of interest

**Output:**
* simple interest = p*t*r
